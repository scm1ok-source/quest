Step 3 · Пошаговый гайд

1) Создайте скрытую переменную

Она будет определять, какой список показывать респонденту.

Пример: переменная age_group

2) Задайте правила через панель логики (Filter → Edit)

Используйте команду SET, чтобы в зависимости от значения q4 задать переменную age_group.

Пример:

SET age_group = 1 IF q4 = 1
SET age_group = 2 IF q4 = 2
SET age_group = 3 IF q4 = 3
SET age_group = 4 IF q4 = 4

3) Создайте динамический список

Теперь сделайте вопрос, где варианты будут зависеть от скрытой переменной age_group.

Пример: Вопрос «Какие бренды напитков вы знаете?»

  • age_group = 1 (до 17 лет): Pepsi, Fanta, Sprite
  • age_group = 2 (18–29 лет): Red Bull, Monster, Burn
  • age_group = 3 (30–44 года): Jacobs, Nescafe, Lipton
  • age_group = 4 (45+): Чай Ahmad, Чай Принцесса Нури, растворимый кофе «Московский»
